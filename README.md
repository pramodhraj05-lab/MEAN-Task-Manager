# MEAN Task Manager

A full-stack Task Manager built with the MEAN Stack as part of the B9IS130 module.

## Tech Stack
- MongoDB - Database
- Express.js - Backend REST API
- Angular - Frontend SPA
- Node.js - Runtime

## How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
ng serve

Open http://localhost:4200 in your browser.

## Features
- Create, view, edit and delete tasks
- Set priority (low, medium, high)
- Set status (pending, in-progress, completed)
- Filter tasks by status and priority
- Due date support
